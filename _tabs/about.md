## Featured project — SOSDash

A conversational analytics agent that answers questions over statistical datasets in plain English and **proves every number with a verifiable, page-referenced citation**. The LLM never touches raw data; it reasons over a metadata contract and retrieves grounded values and evidence from Elasticsearch — which keeps it scalable, data-sovereign, and auditable.
